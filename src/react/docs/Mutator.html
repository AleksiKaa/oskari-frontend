Class: Mutator

Mutator(UIService, functionNames)

The idea behind Mutator is to provide a way to pass multiple functions to an UI component in a single package. We cannot simply pass on an UI Service instance, since those contain functions and properties we don't want to pass to a component. Mutator is a subset of UI service's functions.
