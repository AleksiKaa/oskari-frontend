Features

		A Complete(ly) bundled application
		
Application Classes
		
		-	starter
		-	application
		-	bundle configuration
		-	layers
				
Featured Bundles

		-	enhanced-mapmodule (Map Module Plugins)

		-	trains (GeoRSS, Map Layer, Map Layer, Form) 
		-	solsol (jQuery embedding)
